# `pytest`插件



## allure

- 使用方法
    - 安装allure工具
    - 安装pytest插件
        - `pip install allure-pytest`
        - 注意安装时，所有版本的匹配
    - 执行测试 & 生成报告
        - `pytest test_IC2_test.py --alluredir ./report/allure_raw`
        - `allure serve report/allure_raw`
