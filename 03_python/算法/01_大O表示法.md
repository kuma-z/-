# 大O表示法

> 表示代码运行时间

## 常见大O

- O(log n)，也叫对数时间
    - 这样的算法包括二分查找。
- O(n)，也叫线性时间
    - 这样的算法包括简单查找。
- O(n * log n)，
    - 这样的算法包括快速排序——一种速度较快的排序算法
- O(n^2)
    - 这样的算法包括选择排序——一种速度较慢的排序算法
- O(n!)，
    - 这样的算法包括接下来将介绍的旅行商问题的解决方案——一种非常慢的算法

## 大O的含义

- 算法的速度指的并非时间，而是操作数的增速。
- 谈论算法的速度时，我们说的是随着输入的增加，其运行时间将以什么样的速度增加。
- 算法的运行时间用大O表示法表示。
- O(log n)比O(n)快，当需要搜索的元素越多时，前者比后者快得越多。
